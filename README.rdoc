= estimate_timelog

予定／実績工数比較できるレポーティング機能を提供します。

= 動作確認バージョン

0.8.5

0.8.xであれば、多分動くのではないでしょうか。

= インストール

フォルダをまるごと

vender/plugin

に解凍し、Redmineを再起動するだけでOKです。

dbのmigretionは必要ありません。

= 使い方

トップレベルのメニュー（一番左上）に「予定／実績レポート」のリンクが追加されます。

基本的な使い方は「経過時間」レポートと同様です。

表示されるデータが「実績ベース」ではなく「予定工数ベース」になるだけです。
